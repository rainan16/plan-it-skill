# plan-it

A skill that turns [question-me](https://github.com/rainan16/question-me-skill) session output into a concrete, phased execution plan.

After you've grilled your idea with `question-me`, say **"plan it"** to produce `task_plan.md` with phases, decisions, open questions, and error tracking — ready for execution.

```
Question-me skill grills your idea 
     ↓
Refined intent + key decisions 
     ↓
Plan-it skill 
     ↓
task_plan.md
```

## Quick start

1. Say `/question-me` to clarify your idea (if not present: install [question-me](https://github.com/rainan16/question-me-skill))
2. When it's done, say `/plan-it` — and optionally paste the output
3. Plan-it creates a phased `task_plan.md` with decisions table and error log
